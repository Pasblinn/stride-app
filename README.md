# Stride App

Aplicativo de registro de atividades físicas desenvolvido em Flutter, inspirado no Strava.

## Sobre
Projeto da disciplina de Dispositivos Móveis - UTFPR.

## Estrutura do Projeto
O projeto segue o padrão MVC (Model-View-Controller) com Provider para gerenciamento de estado.

```
lib/
├── main.dart
├── models/
├── controllers/
├── repositories/
└── views/
```

## Como Executar
1. Certifique-se de ter o Flutter instalado (versão 3.41+)
2. Clone o repositório
3. Execute `flutter pub get`
4. Execute `flutter run`
